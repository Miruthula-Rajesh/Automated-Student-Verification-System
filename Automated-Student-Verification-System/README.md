# 📧 Automated Student Verification System

A Flask-based web application that automates student verification using a student database.

## Features

- Student Registration
- Student Verification
- Verification Logs
- Search Students
- SQLite Database
- Responsive Dashboard
- Delete Students
- Verification History

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript

## Installation

```bash
pip install -r requirements.txt
python database.py
python app.py
```
